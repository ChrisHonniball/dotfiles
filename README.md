# My DotFiles

This (like other dotfiles) repositories are all of my dotfiles for setting on my computer.

## Cloneing

```
cd ~
git clone --recursive git@github.com:ChrisHonniball/dotfiles.git dotfiles
```

## Installation

```
cd ~/dotfiles
./install.sh
```

## Uninstallation

```
cd ~/dotfiles
./uninstall.sh
```
